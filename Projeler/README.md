# Projeler

- Bu klasörde katılımcılardan oluşturulan proje klasörleri barındırır.

# Proje Ekleme Kuralları

1. Projenin adı istenildiği gibi adlandırılabilir, serbersttir.
2. Her projenin **Mutlaka** README.md dosyası olmalıdır.
3. Projelerin README.md dosyasında proje hakkında anlaşılır bir dille açıklanmalıdır.
4. Projelere katılım tamamen serbesttir.
