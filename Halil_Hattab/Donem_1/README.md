# Halil_Hattab - 1. Dönem
- bu klasör [Halil Hattab (trs-1342)](https://github.com/trs-1342) tarafından 1. Dönemde geliştirilmiştir.

## Klasör Yapısı

1. HelloWorld.c:
	- ilk yazılan dosya, evrensel mesajı printf ile vermektedir.

2. if_else.c:
	- önceden tanımlanmış değişkenlerden not ortalamasını hesaplayarak öğrencinin sınıfta kalıp kalmadığını hesaplar.

3. scanf.c:
	- `scanf` fonksiyonunun nasıl çalıştığını anlatır.

4. HesapMakinesi.c:
	- fonksiyonlu hesap makinesi. (_açıkları var_)

5. start.c:
	- örnek c programlama dosyası.


> [!NOTE]
> Bu proje geliştirilme sürecindedir, genel olarak hafta sonları aktif katılım sağlayabiliyorum.
